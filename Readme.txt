Portfolio Website
A modern, fully responsive portfolio website built with HTML, CSS, Bootstrap, and JavaScript. This project highlights my skills, projects, and achievements, offering an interactive and user-friendly experience. The design focuses on clean aesthetics, smooth navigation, and performance across various devices.

Features:
Responsive Layout: Optimized for desktops, tablets, and mobile devices.
Modern Design: Clean UI/UX with smooth transitions and interactive elements.
Project Showcase: Displays key projects with links to GitHub repositories.
Contact Form: Allows users to reach out directly through the contact page.
GitHub Integration: Direct links to my GitHub profile and repositories.
The website provides a clear overview of my professional work and serves as a platform for potential employers, collaborators, and clients to explore my projects and skills.

Technologies Used:
HTML5 for structure
CSS3 and Bootstrap for responsive design
JavaScript for interactivity
Icons from Bootstrap Icons
The site is hosted on GitHub Pages, making it easily accessible for visitors. It serves as an example of my work in front-end development, with a focus on creating polished, user-friendly interfaces.
